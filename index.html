Minhas Skills

HTML

O HTML é o responsável por todo o conteúdo da página. É através dele que todos os textos, imagens, links, listas, tabelas são inseridos no site. Através dele, dizemos ao navegador o que cada item da página significa.

CSS

O CSS fica com a estilização do site. É ele quem de fato define a aparência. Cores, tipografia, layout, tudo isso fica a cargo do CSS. Sem ele, os sites seriam apenas uma sequência de textos e imagens, sem nenhum estilo nem identidade própria.

JavaScript

Esse aqui, por sua vez, é quem deixa o site realmente “moderno”. É o responsável pela interatividade. Sem ele, os sites seriam estáticos, ou seja, você iria apenas olhar para eles, sem interagir de fato. Sem ele, você não conseguiria nem deixar um like nas fotos dos seus amigos.
